# /quit 命令

退出 PilotCode 程序。

## 作用

- 正常退出 PilotCode
- 保存会话状态（如果配置）
- 关闭与 LLM 的连接

## 基本用法

```bash
/quit
```

## 别名

以下命令效果相同：

```bash
/quit
/exit
/q
```

## 退出行为

1. **保存配置**：自动保存当前设置
2. **会话持久化**：如果启用了会话保存，会自动保存
3. **清理资源**：关闭网络连接、释放内存

## 使用场景

### 场景1：正常退出

```bash
# 完成工作后退出
/quit
```

### 场景2：快速退出

```bash
# 使用快捷键
/q
```

## 与 /clear 的区别

| 命令 | 作用 | 程序状态 |
|------|------|----------|
| `/quit` | 退出程序 | 程序结束 |
| `/clear` | 清屏 | 继续运行 |

## 快捷键

- `Ctrl+C` - 中断当前操作
- `Ctrl+D` - 退出（Unix/Linux）
- `/quit` - 斜杠命令

## 注意事项

1. **未保存的工作**：退出前确保重要修改已保存
2. **后台任务**：如果有进行中的后台任务，可能会提示确认
3. **会话恢复**：下次启动时可以恢复会话（如果配置了持久化）

## 相关命令

- `/clear` - 清屏（不退出）
- `/session` - 会话管理
