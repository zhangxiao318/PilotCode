# /help 命令

显示帮助信息，列出所有可用命令及其简要说明。

## 作用

- 查看所有可用的斜杠命令
- 了解每个命令的基本功能
- 快速获取命令列表

## 基本用法

```bash
/help
```

## 输出示例

```
可用命令：
  /help        - 显示帮助信息
  /index       - 索引代码库
  /search      - 搜索代码
  /config      - 配置管理
  /model       - 模型设置
  /theme       - 切换主题
  /session     - 会话管理
  /cost        - 用量统计
  /clear       - 清除屏幕
  /quit        - 退出程序

输入 /help <command> 查看详细帮助
```

## 查看特定命令帮助

```bash
/help <command>
```

例如：
```bash
/help index
/help search
```

## 使用场景

### 场景1：忘记命令时

```bash
# 查看所有可用命令
/help
```

### 场景2：了解特定命令用法

```bash
# 查看 /index 命令的详细帮助
/help index
```

## 相关命令

所有其他命令的帮助都可以通过 `/help` 查看。
