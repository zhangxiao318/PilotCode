# /clear 命令

清除屏幕上的所有内容，提供一个干净的工作界面。

## 作用

- 清除终端屏幕上的所有历史消息
- 重置显示区域
- 保持会话上下文不变（仅清除显示）

## 基本用法

```bash
/clear
```

## 特点

- **仅清除显示**：不影响会话历史、变量状态等
- **保留上下文**：AI 仍然记得之前的对话
- **快捷键支持**：通常可以使用 `Ctrl+L` 达到相同效果

## 使用场景

### 场景1：屏幕内容过多

```bash
# 经过长时间对话后，屏幕内容太多
/clear
```

### 场景2：开始新任务

```bash
# 完成一个任务，准备开始新的
/clear
# 现在屏幕干净，开始新任务
```

### 场景3：误操作后清理

```bash
# 不小心输错了命令，屏幕显示混乱
/clear
```

## 与 /quit 的区别

| 命令 | 作用 | 会话状态 |
|------|------|----------|
| `/clear` | 清除屏幕显示 | 保持运行 |
| `/quit` | 退出程序 | 结束会话 |

## 快捷键

- `Ctrl+L` - 清屏（大多数终端支持）
- `/clear` - 斜杠命令方式

## 相关命令

- `/quit` - 退出程序
- `/session` - 会话管理
